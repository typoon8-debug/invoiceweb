# 노션 견적서 웹 뷰어 MVP PRD

## 🎯 핵심 정보

**목적**: 노션에서 작성한 견적서를 클라이언트가 별도 로그인 없이 웹에서 확인하고 PDF로 저장할 수 있게 한다
**사용자**: 프리랜서·소규모 사업자(견적서 작성자)와 그 클라이언트(견적서 수신자)

---

## 🚶 사용자 여정

```
[작성자 여정]

1. 로그인 페이지
   ↓ 노션 OAuth 로그인

2. 대시보드 페이지
   ↓ "새 견적서" 버튼 클릭

3. 견적서 생성 페이지
   ↓ 노션 데이터베이스에서 견적 항목 불러오기 + 고객 정보 입력

4. 견적서 미리보기 페이지
   ↓ 확인 후 "공유 링크 생성" 클릭

5. 공유 링크 생성 완료
   ↓ 링크 복사 → 클라이언트에게 전달

---

[클라이언트 여정]

1. 공유 링크 접속 (토큰 포함 URL)
   ↓ 토큰 유효성 검사

   [유효] → 견적서 뷰어 페이지
   [만료/무효] → 링크 만료 안내 페이지

2. 견적서 뷰어 페이지
   ↓ 내용 확인 후 "PDF 다운로드" 클릭

3. PDF 다운로드 완료
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID       | 기능명           | 설명                                                            | MVP 필수 이유                        | 관련 페이지                                |
| -------- | ---------------- | --------------------------------------------------------------- | ------------------------------------ | ------------------------------------------ |
| **F001** | 노션 데이터 연동 | 노션 데이터베이스에서 견적 항목(품목, 수량, 단가) 자동 불러오기 | 핵심 가치: 노션을 입력 도구로 활용   | 견적서 생성 페이지                         |
| **F002** | 견적서 생성      | 고객 정보 + 노션 항목 조합으로 견적서 문서 생성                 | 서비스 핵심 기능                     | 견적서 생성 페이지, 견적서 미리보기 페이지 |
| **F003** | 공유 링크 생성   | 만료 기한이 있는 고유 토큰 URL 생성                             | 클라이언트 접근의 유일한 수단        | 견적서 미리보기 페이지                     |
| **F004** | 견적서 웹 뷰어   | 토큰 URL로 접속 시 견적서를 웹 페이지로 렌더링                  | 클라이언트가 내용 확인하는 핵심 기능 | 견적서 뷰어 페이지                         |
| **F005** | PDF 다운로드     | 뷰어 페이지에서 견적서를 PDF 파일로 저장                        | 클라이언트의 핵심 니즈(보관·출력)    | 견적서 뷰어 페이지                         |
| **F006** | 견적서 목록 관리 | 작성한 견적서 목록 조회, 공유 링크 상태(활성/만료) 확인         | 작성자의 기본 관리 기능              | 대시보드 페이지                            |

### 2. MVP 필수 지원 기능

| ID       | 기능명          | 설명                                       | MVP 필수 이유                   | 관련 페이지                    |
| -------- | --------------- | ------------------------------------------ | ------------------------------- | ------------------------------ |
| **F010** | 기본 인증       | 회원가입/로그인/로그아웃 (이메일+비밀번호) | 작성자 계정 보호 및 데이터 격리 | 로그인 페이지, 회원가입 페이지 |
| **F011** | 노션 OAuth 연결 | 작성자 계정에 노션 워크스페이스 연결       | F001 작동을 위한 필수 선행 조건 | 대시보드 페이지                |

### 3. MVP 이후 기능 (제외)

- 견적서 템플릿 커스터마이징 (로고, 색상 테마)
- 견적서 승인/반려 상태 관리
- 클라이언트 이메일 자동 발송
- 견적서 수정 이력 관리
- 다중 통화 지원
- 알림 시스템 (링크 열람 알림 등)

| **사용자 행동** | 노션 데이터베이스 선택 → 항목 확인/수정 → 고객 정보 입력 → 미리보기 이동                                                                                                                                                        |

## 🗄️ 데이터 모델

| 필드              | 설명            | 타입/관계 |
| ----------------- | --------------- | --------- |

| notionAccessToken | 노션 OAuth 토큰 | string    |
| createdAt         | 가입일          | timestamp |

| 필드           | 설명                | 타입/관계 |
| -------------- | ------------------- | --------- |
| id             | 고유 식별자         | UUID      |

| clientName     | 클라이언트 회사명   | string    |
| clientContact  | 클라이언트 담당자명 | string    |
| validUntil     | 견적 유효기간       | date      |
| totalAmount    | 합계 금액           | number    |
| shareToken     | 공유 링크 토큰      | string    |
| tokenExpiresAt | 토큰 만료일시       | timestamp |
| createdAt      | 생성일              | timestamp |
